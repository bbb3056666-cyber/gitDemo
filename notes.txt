几种状态:untracked未跟踪,modified已修改,staged已暂存,clean干净状态(已提交且没修改)

新文件->untracked
git add->staged
git commit->tracked(clean)
修改文件->modified
git add->staged
git commit->tracked(clean)


命令:
git init ===============================初始化仓库
git status
git add .  ==============================把所有文件加入暂存区
git commit -m "本次提交说明"=============提交,新增一个版本,-m message
git commit --amend -m "新的message"=====用暂存区最新内容覆盖最近一次提交 + 更新提交说明
